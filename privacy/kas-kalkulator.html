Kebijakan Privasi

Kas Kalkulator • Terakhir diperbarui: Desember 2024
Ringkasan: Kas Kalkulator adalah aplikasi offline yang tidak mengumpulkan atau mengirim data pribadi Anda ke server manapun.

1. Prinsip Privasi

Kas Kalkulator dirancang dengan prinsip "Privacy by Design". Aplikasi ini:

  • Bekerja sepenuhnya offline
  • Tidak memerlukan koneksi internet
  • Tidak mengumpulkan data pribadi
  • Tidak menggunakan tracking atau analytics

2. Data yang Disimpan

Aplikasi hanya menyimpan data yang Anda masukkan secara lokal:

  • Transaksi kas harian
  • Kategori pemasukan dan pengeluaran
  • Pengaturan aplikasi (mata uang, format tanggal)
  • Laporan yang Anda buat

3. Tidak Ada Pengumpulan Data

Kami TIDAK mengumpulkan:

  • Informasi identitas (nama, email, nomor telepon)
  • Data keuangan atau bisnis Anda
  • Informasi lokasi
  • Data penggunaan aplikasi
  • Informasi perangkat

4. Izin Aplikasi

Aplikasi meminta izin minimal:

  • Penyimpanan: Untuk menyimpan data transaksi dan laporan
  • Export File: Untuk mengekspor laporan ke PDF (opsional)

5. Keamanan Data

Data Anda aman karena:

  • Disimpan hanya di perangkat Anda
  • Tidak ada transmisi data ke internet
  • Akses terbatas pada aplikasi saja
  • Enkripsi penyimpanan lokal

6. Export dan Backup

Fitur export laporan:

  • File PDF disimpan di penyimpanan lokal
  • Anda yang mengontrol penuh file tersebut
  • Tidak ada upload otomatis ke cloud

7. Tidak Ada Iklan

Aplikasi ini tidak menampilkan iklan dan tidak menggunakan SDK periklanan yang dapat mengumpulkan data.

8. Update Aplikasi

Update aplikasi hanya untuk:

  • Perbaikan bug
  • Penambahan fitur
  • Peningkatan keamanan

Tidak ada perubahan pada kebijakan pengumpulan data karena kami tidak mengumpulkan data.

9. Kontak

Untuk pertanyaan tentang privasi:

Email: privacy@nwdev.com

Jaminan: Kas Kalkulator berkomitmen untuk menjaga privasi Anda dengan tidak mengumpulkan data apapun. Aplikasi ini dirancang untuk bisnis kecil yang mengutamakan keamanan data keuangan.
