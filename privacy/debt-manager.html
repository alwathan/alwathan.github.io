Kebijakan Privasi

Debt Manager • Terakhir diperbarui: Desember 2024
Ringkasan: Debt Manager tidak mengumpulkan, menyimpan, atau membagikan data pribadi Anda. Semua data disimpan secara lokal di perangkat Anda.

1. Informasi yang Kami Kumpulkan

Debt Manager TIDAK mengumpulkan informasi pribadi apapun dari pengguna, termasuk:

  • Data identitas pribadi (nama, email, nomor telepon)
  • Informasi keuangan atau data hutang
  • Data lokasi atau informasi perangkat
  • Riwayat penggunaan aplikasi

2. Penyimpanan Data

Semua data yang Anda masukkan ke dalam aplikasi Debt Manager disimpan secara lokal di perangkat Android Anda. Data ini meliputi:

  • Daftar hutang dan piutang
  • Catatan pembayaran
  • Pengaturan aplikasi

Data ini tidak dikirim ke server eksternal atau cloud storage manapun.

3. Keamanan Data

Kami mengimplementasikan langkah-langkah keamanan untuk melindungi data Anda:

  • Enkripsi data lokal
  • Tidak ada koneksi internet yang diperlukan
  • Akses data hanya melalui aplikasi

4. Berbagi Data

Debt Manager TIDAK PERNAH membagikan data Anda dengan pihak ketiga karena kami tidak mengumpulkan data apapun.

5. Izin Aplikasi

Aplikasi ini meminta izin minimal:

  • Penyimpanan: Untuk menyimpan data aplikasi dan backup lokal
  • Notifikasi: Untuk reminder pembayaran (opsional)

6. Backup dan Restore

Fitur backup hanya menyimpan data di penyimpanan lokal perangkat Anda. Anda dapat memilih untuk menyimpan backup di penyimpanan eksternal (SD card) namun data tetap tidak dikirim ke internet.

7. Perubahan Kebijakan

Jika ada perubahan pada kebijakan privasi ini, kami akan memperbarui aplikasi dan memberitahu pengguna melalui update aplikasi.

8. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, silakan hubungi kami di:

Email: privacy@nwdev.com

Catatan Penting: Karena semua data disimpan lokal, pastikan Anda membuat backup secara berkala untuk menghindari kehilangan data jika perangkat rusak atau hilang.
