Kebijakan Privasi

Terakhir diperbarui: Desember 2024

1. Informasi yang Kami Kumpulkan

Kas Kalkulator tidak mengumpulkan atau menyimpan data pribadi pengguna. Aplikasi berfungsi secara offline tanpa koneksi internet.

2. Penggunaan Data

  • Semua perhitungan dilakukan secara lokal di perangkat
  • Tidak ada data yang dikirim atau disimpan di server
  • Riwayat perhitungan (jika ada) hanya tersimpan sementara di memori aplikasi

3. Keamanan Data

Karena tidak ada data yang dikumpulkan atau disimpan, tidak ada risiko kebocoran data pribadi.

4. Izin Aplikasi

Aplikasi ini tidak memerlukan izin khusus dan dapat berjalan tanpa akses ke data sensitif perangkat.

5. Kontak

Jika ada pertanyaan tentang kebijakan privasi ini, hubungi kami di: contact@nwdev.com
