Kebijakan Privasi

Terakhir diperbarui: Desember 2024

1. Informasi yang Kami Kumpulkan

CashFlow akan mengumpulkan data transaksi keuangan yang Anda input untuk memberikan analisis arus kas yang akurat.

2. Penggunaan Data

  • Data transaksi digunakan untuk analisis dan laporan arus kas
  • Data disimpan secara lokal dengan opsi sinkronisasi cloud (opsional)
  • Tidak ada data yang dibagikan kepada pihak ketiga

3. Keamanan Data

Data keuangan Anda akan dienkripsi dan dilindungi dengan standar keamanan tinggi. Akses data hanya melalui autentikasi pengguna.

4. Izin Aplikasi

  • Penyimpanan: Untuk menyimpan data transaksi
  • Internet: Untuk fitur sinkronisasi (opsional)
  • Notifikasi: Untuk pengingat dan laporan berkala

5. Hak Pengguna

  • Menghapus semua data kapan saja
  • Mengekspor data dalam format yang dapat dibaca
  • Menonaktifkan fitur sinkronisasi cloud
